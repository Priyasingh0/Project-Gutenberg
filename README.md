# Project Gutenberg

Project Gutenberg is a modern book discovery and audiobook web application built using React.js.  
The platform allows users to explore books, browse categories, manage wishlists, and interact with different sections through a clean and responsive interface.

---

## 🚀 Features

- 📚 Browse books by categories
- 🎧 Audiobook section
- ❤️ Wishlist management
- 👤 User profile system
- 🔎 Search functionality
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- Tailwind CSS
- HTML5
- CSS3

---

## ⚙️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Priyasingh0/Project-Gutenberg.git
  
2. Navigate to the project folder:
   cd Project-Gutenberg

3. Install dependencies:
   npm install

4. Start the development server:
   npm start
